## ✨ Возможности

### 🎨 Генерация фото

| Модель | Описание | Время |
|--------|----------|-------|
| **GPT Image 1.5** | Универсальный флагман от OpenAI | 15-45 сек |
| **Nano Banana Pro** | Фотореализм, скорость 3-5 секунд | 3-5 сек |
| **FLUX.2 flex** | Идеальный рендеринг текста, логотипы | 10-30 сек | 
| **Умный выбор** | Бот сам подбирает модель под задачу | — | 

### 🖼️ Редактирование фото
- Изменение фона, стиля, объектов
- Добавление текста на изображения
- «Оживление» фото (улучшение качества)

### 🔄 Объединение фото
- Сбор до 16 фото в одну композицию
- Интеллектуальное размещение объектов
- Произвольный фон и стиль

### 🎬 Генерация видео

| Модель | Длительность | Особенность |
|--------|--------------|-------------|
| **Wan-video** | 5 сек | Быстрая генерация (30-60 сек) |
| **Kling 3.0 Omni** | 5-15 сек |Киношное качество + синхронный звук |
| **Hailuo 2.3** | 6-10 сек | Баланс цены и качества |

**Режимы Kling:**
- Text-to-Video — создание видео из текста
- Image-to-Video — анимация вашего фото
- Video-to-Video — преобразование стиля видео

- Логирование всех транзакций
- Бонусная система (+50 ₽ за подписку)

### 👑 Админ-панель
- Просмотр статистики генераций
- Изменение всех цен (фото, видео)
- Просмотр логов ошибок

## 🔧 Настройка

### 🔑 API ключи
###  1. MAX Bot Token
1. Перейдите на https://max.ru/bots
2. Создайте нового бота
3. Скопируйте токен
```dotenv
MAX_BOT_TOKEN=ваш_токен
MAX_BOT_ID=id_бота
```
### 2. Replicate API Token
1. Зарегистрируйтесь на https://replicate.com
2. Account → API Tokens → Create token
```dotenv
REPLICATE_API_TOKEN=r8_...ваш_токен
```

### 3. ЮKassa (платежи)
1. https://yookassa.ru → Регистрация
2. Получите shop_id и секретный ключ
```dotenv
YOOKASSA_SHOP_ID=ваш_id
YOOKASSA_SECRET_KEY=ваш_ключ
```

### 4. VK API (бонусная система)
1. Создайте сообщество VK
2. Управление → Работа с API → Создать ключ
```dotenv
VK_BOT_TOKEN=vk1...ваш_токен
VK_GROUP_ID=id_группы
```
